##  그룹웨어 프로그램 - GURUBEE


### 📌 담당 기능: 전자 결재 기능, DB모델링, 메인화면 구현
##### ✔ 전자결재 주요 기능
- 전자결재 문서 발신, 수신함 조회
- 결재 문서 임시보관함 조회
- 결재 문서 유형별 폼 출력
- 결재 라인 지정 기능
- 결재 문서 등록, 수정
- 임시보관 문서 등록, 삭제
- 결재 문서 승인, 반려 기능
- 다중 파일 업로드 기능

<br>

- Servlet & JSP
